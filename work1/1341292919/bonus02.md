**一、切片与数组的区别**

1.长度：

数组的长度是固定的，一旦声明，长度不能改变。

切片的长度是动态的，可以按需增长或缩小。

2.传递方式

数组：数组作为参数传递时，会进行值拷贝，传递的是整个数组的副本。

```
func modifyArray(arr [5]int) {
    arr[0] = 100
}
```

切片：切片作为参数传递时，传递的是切片的引用，修改切片会影响到底层数组。

```
func modifySlice(slice []int) {
    slice[0] = 100
}
```

3.声明方式

切片不需要声明长度，数组需要声明长度

4.内存分配

数组：数组在声明时会分配一块连续的内存空间。

切片：切片是对底层数组的一个动态视图，它包含一个指向底层数组的指针、长度和容量。

**二、创建切片的方式：**

1.通过数组创建切片

`arr := [5]int{1, 2, 3, 4, 5}` 

`slice1 := arr[1:4]`

2.直接创建切片

`slice2 := []int{6, 7, 8}`

3.使用make函数创建切片

`slice3 := make([]int, 3, 5)`

(向切片中添加元素则可以通过调用append函数)

**三、map**

map:其包含的元素是一对键值对（两个值相对映照）

map中的元素是无序的，这意味着当你遍历map时，元素的顺序是不确定的。

map的大小是动态的，可以根据需要添加或删除元素。

创建方式：

1.使用make函数

`m := make(map[string]int)`

2.直接声明创造

```
m := map[string]int{
    "apple": 1,
    "banana": 2,
}
```
