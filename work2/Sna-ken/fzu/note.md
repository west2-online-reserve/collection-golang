**基本步骤**

1.构造客户端

2.创建请求

3.发送请求

4.解析网页

5.存进SQL或者打印

**如果要发送多次请求不用重复构造客户端，可以在函数中引入**

**遇到JavaScript动态内容处理方法**

打开浏览器开发者工具（按 F12）。

进入 Network （网络）选项卡。

刷新页面或进行能触发数据加载的操作。

在请求列表中，寻找类型为 XHR 或 Fetch 的请求。

仔细观察这些请求的 Response（响应），看其中是否包含一个 JSON 格式的数据，里面就有 clickCount, hits, visit 或直接是 3221 这样的字段。

如果找到了这样的接口，爬虫就可以直接模拟请求这个接口 URL，从而轻松地拿到结构化的数据（包括点击量）
